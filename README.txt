+ Các bước chạy project bằng eclipse:

- Đầu tiên import folder Final trong eclipse
- Sau đó vào file application.properties 
- Cuối cùng điền thông tin sql (db.datasource.username và db.datasource.password) đúng với cấu hình của máy

+ Để chạy dữ liệu Database ta cần tạo database tên finalproject bằng MySQL và thêm các dữ liệu đã có sẵn trong folder db đã cung cấp sẵn
